Confidence-Aware ProSST for Zero-Shot Mutation Effect Prediction on ProteinGym

Posted on 2026-01-01 Edited on 2026-07-19 In Archive

Abstract

ProSST (NeurIPS 2024) improves protein mutation effect prediction by adding a quantized structural channel to a protein language model, but it treats all structural tokens as equally reliable. This project asks whether residue-level AlphaFold pLDDT can improve zero-shot scoring on ProteinGym by modulating that channel at inference time or at score level. We study three embedding-side variants: hard masking below a pLDDT threshold, soft scaling by pLDDT/100, and a learned sigmoid gate trained with a frozen-backbone masked-language-model objective. On 217 ProteinGym assays, our ProSST-2048 reproduction reaches mean Spearman 0.5225 (95% CI [0.500, 0.545]), slightly above the paper’s reported 0.504. None of the direct confidence-aware variants improves over this baseline: hard masking at τ = 50/70/90 yields 0.5203 / 0.5177 / 0.4832, soft weighting yields 0.5203, and learned gating yields 0.5218. We then evaluate score-level confidence conditioning on the 214-assay intersection where ProSST-128, ProSST-512, ProSST-2048, and ESM-2 scores are all available. A flat stacked ensemble reaches 0.5569, and a confidence-conditioned gated ensemble further improves this to 0.5631 (95% CI [0.542, 0.585]). The largest gain appears in the lowest-confidence quartile, which rises from 0.4918 to 0.5138. These results suggest a sharper conclusion: simple residue-wise confidence modulation does not improve ProSST at the embedding stage, but confidence is still useful when it controls how different predictors are combined at the assay level.

1. Introduction

1.1 Problem and Motivation

Structure-aware protein language models often outperform sequence-only models for mutation effect prediction because they can condition on local geometry in addition to sequence context. ProSST is a recent example: it discretizes local structure into tokens and injects them through disentangled sequence–structure attention. The practical weakness of this setup is that predicted structures are not uniformly reliable. AlphaFold’s pLDDT varies considerably across residues and proteins, especially in flexible or poorly constrained regions.

This motivates a natural question: if structural confidence is already available, can we use it to improve ProSST’s zero-shot mutation scores? The idea is attractive because it promises a lightweight improvement without retraining the full backbone. It is also scientifically useful even if it fails: if pLDDT-aware modulation does not help, that tells us something non-trivial about how ProSST already encodes structure reliability.

1.2 Why The Problem Is Difficult

The challenge is not just to add confidence, but to add it at the right level. pLDDT is a coarse scalar confidence proxy, while ProSST’s structure vocabulary and attention machinery are already learned representations of local geometry. An overly simple rule may overwrite useful signal rather than refine it. Negative results are also easy to misread unless the baseline is reproduced on the same assay set and statistical uncertainty is reported.

1.3 Contributions

  • We implement three confidence-aware ProSST variants that modulate the shared structure embedding tensor before the encoder.
  • We reproduce ProSST-2048 on 217 ProteinGym assays with bootstrap confidence intervals and pLDDT-quartile analysis.
  • We show that all three direct pLDDT-aware variants fail to improve zero-shot performance, with aggressive masking causing large regressions.
  • We introduce a confidence-conditioned score-level ensemble that uses pLDDT to gate the relative contribution of multiple predictors and improves over a flat stacked ensemble on the 214-assay overlap.
  • We document an evaluation protocol that treats sequence–confidence alignment as a hard requirement, preventing silent distortion of confidence-aware inputs.

2. Related Work / Background

2.1 Protein Language Models

Sequence-only protein language models such as ESM-1b and ESM-2 learn from large protein corpora with masked-language modeling. Structure-aware models extend this by injecting predicted or derived structural information. SaProt couples each amino acid with a Foldseek 3Di token. ProSST instead learns a discrete structure vocabulary through vector quantization and uses disentangled attention between sequence and structure streams [1]. These approaches typically assume that the structure channel is always trustworthy.

2.2 Structural Confidence

AlphaFold’s pLDDT is the standard per-residue confidence estimate for predicted structures [3, 4]. It is commonly used for filtering low-confidence regions, but it is less clear whether it can be profitably fused into a pretrained structure-aware language model at inference time. This project tests exactly that narrow but meaningful question.

3. Approach

3.1 Baseline: ProSST

ProSST embeds amino-acid tokens and structure tokens into two hidden-state streams. A key architectural detail is that the structure embedding tensor is created once and then reused by every encoder layer through disentangled sequence–structure attention. This makes the embedding stage a natural intervention point: one transformation affects all layers while preserving the original backbone.

3.3 Confidence-Conditioned Score-Level Ensemble

The embedding-side interventions above test whether pLDDT can improve a single pretrained structure-aware model. We also study a different hypothesis: confidence may be more useful for deciding which predictor to trust than for directly rewriting a predictor’s internal representation. To test this, we combine five score channels available on the shared 214-assay subset: the structure streams from ProSST-128, ProSST-512, and ProSST-2048, the sequence-only stream from ProSST-2048, and ESM-2-650M.

3.4 Implementation Details

  • The model is implemented as a lightweight extension of the public ProSST masked-language model, so the pretrained backbone can be reused directly while only the confidence-aware components are newly added.
  • V3 uses a frozen-backbone masked-LM objective on 173 held-out proteins, with 44 validation proteins, 3 epochs, batch size 1, learning rate 10−310^{-3}10−3, and fp16.
  • The main evaluation path uses vectorized mutant aggregation to avoid large numbers of GPU→CPU synchronizations on high-mutation assays.
  • To prevent confidence misalignment artifacts, sequence–structure mismatches are treated as explicit data-quality problems rather than being silently repaired by truncation or padding.
  • For the score-level ensemble, we cache per-mutant logit differences for each channel once and perform all weight selection on those cached scores. This separates expensive model inference from lightweight ensemble search and makes confidence-gated reweighting reproducible across folds.

4. Experiments

4.1 Data

We evaluate on the 2024 ProteinGym substitution split: 217 assays and roughly 1.5M mutations [2]. Each assay includes a wild-type residue FASTA, a ProSST structure-token FASTA, and a substitutions CSV with experimental DMS scores. pLDDT arrays are extracted from AlphaFoldDB PDB files and aligned to the residue FASTA.

The evaluation protocol enforces exact residue-level alignment between the wild-type sequence and the pLDDT array used for confidence-aware scoring. Any sequence/PDB inconsistency is treated as a data-quality issue that must be resolved before evaluation. This design choice is important for the present study because even small alignment shifts would change which residues are masked or down-weighted, directly confounding the interpretation of the confidence-aware ablations.

For the ensemble analysis, we additionally require that all five score channels be present for the same assay. This yields a 214-assay intersection. The ensemble numbers are therefore reported separately from the 217-assay direct modulation results rather than mixed into a single table.

4.2 Evaluation Metrics

The primary metric is per-assay Spearman’s ρ, averaged across assays. We also report NDCG and top-10% recall. Statistical uncertainty is estimated by bootstrap over assays with n=1000n = 1000n=1000 resamples. To test whether confidence-aware methods help specifically on uncertain proteins, we also stratify assays into four quartiles by mean pLDDT.

4.3 Experimental Settings

All reported CA-ProSST runs use the public ProSST-2048 model with the backbone frozen. Full 217-assay evaluation takes about 40 minutes per variant with the vectorized scorer on H100 hardware. V3 model selection is based on validation loss.

5. Results

Two points stand out in Table 1. First, the baseline reproduction is solid: 0.5225 is slightly above the published ProSST score of 0.504 and its confidence interval comfortably covers that paper number. Second, none of the confidence-aware variants beats baseline. The learned gate is closest, but the difference is only −0.0007 in mean Spearman and is well inside the bootstrap uncertainty range. Hard masking gets worse as the threshold becomes more aggressive, with τ = 90 causing a large drop of −0.0393.

The quartile breakdown in Table 2 confirms that confidence matters at the dataset level: low-confidence proteins are substantially harder than high-confidence proteins under every model. However, direct confidence modulation does not close that gap. The V3 gate is nearly identical to baseline in all quartiles, while the hard-mask variants become progressively worse as more structure is removed.

Table 3 changes the interpretation of the project. The direct embedding-side variants remain negative, but confidence is not useless overall. When it is used to modulate the mixture of heterogeneous predictors rather than to suppress structure embeddings residue by residue, it produces a measurable gain. The flat ensemble already improves strongly over any single channel, reaching 0.5569 mean Spearman on the 214-assay overlap. Adding confidence-conditioned gating raises this further to 0.5631, a gain of +0.0062 over the flat ensemble and +0.0410 over the ProSST-2048 single-channel score on the same overlap. The oracle comparison shows the same pattern: flat oracle Spearman is 0.5574, while gated oracle Spearman is 0.5639.

The confidence-stratified behavior of the gated ensemble is also informative. Relative to the flat ensemble, the largest gain occurs in the lowest-confidence quartile (0.4918 → 0.5138, +0.0219), while the middle and upper quartiles change only modestly. This is exactly the regime where naive hard masking caused the greatest damage. Confidence therefore appears useful not as a binary filter on structure tokens, but as a signal for how aggressively to trust structurally grounded versus complementary predictors.

6. Analysis

6.1 Per-Assay Gain/Loss Distribution

The learned gate is best understood as a near-baseline model rather than an improvement. Compared with baseline, it improves 83 of the 217 assays and degrades 134. The mean gain on improved assays is only +0.0028 Spearman, while the mean loss on degraded assays is −0.0029. The closeness of these two numbers explains why the aggregate score remains essentially unchanged: the gate learns a weak modulation that neither consistently rescues low-confidence proteins nor consistently harms high-confidence ones.

6.2 What The Ensemble Learns From Confidence

The score-level ensemble behaves differently from the embedding-side gate. On the 214-assay overlap, the gated ensemble improves 139 assays and degrades 75 relative to the flat ensemble, with mean per-assay gain +0.0062. The best gains are concentrated in difficult, lower-confidence assays, while the largest losses are smaller in aggregate and are not confined to a single pLDDT regime. This asymmetric gain/loss pattern explains why the overall mean rises even though the quartile averages outside Q1 move only slightly.

The fitted gating parameters are also interpretable. Across folds, the dominant weight remains on the ProSST-2048 structure channel, but the learned operating points for most structural channels concentrate near mean pLDDT 55 with a relatively smooth temperature of 20. This indicates that the model does not prefer abrupt switching; instead it performs gradual confidence-aware rebalancing. The sequence-only ProSST channel is consistently driven to zero weight, while ESM retains a non-zero contribution and occasionally receives a higher operating point. Empirically, the gated ensemble is therefore not discovering a new backbone-level representation. It is discovering a more effective allocation rule for when to emphasize the strongest structural channel and when to preserve support from a non-structural model.

6.3 Threshold Sensitivity By Confidence Regime

The hard-mask ablation provides a clearer diagnostic. Raising the threshold from 50 to 70 to 90 produces a monotonic decline in overall mean Spearman (0.5203 → 0.5177 → 0.4832), but the damage is highly concentrated in lower-confidence proteins. Under the most aggressive setting, Q1 falls from 0.4490 to 0.3645 and Q2 falls from 0.4982 to 0.4370, whereas Q4 is almost unchanged (0.5744 to 0.5742). This pattern directly contradicts the naive rule that low pLDDT residues should simply be discarded. Even when the predicted structure is imperfect, ProSST still extracts useful signal from it.

6.4 Why The Learned Gate Still Fails

The learned gate also clarifies why additional flexibility is not sufficient by itself. Training loss decreases from 2.2794 to 2.1872 before rising again, and the best validation loss is obtained at epoch 1 (2.0936), followed by a slight regression at epoch 2 (2.1635). This indicates that the gate can fit the auxiliary masked-language-model objective, but only weakly. More importantly, that objective does not directly supervise mutation-effect ranking. Because the backbone is frozen, the gate can only rescale an already-trained structural representation; it cannot relearn how structural confidence should interact with fitness prediction. The empirical result is therefore conservative behavior: the gate remains very close to the pretrained baseline and produces no reliable improvement.

7. Conclusion

This project asked a focused question: can AlphaFold confidence improve zero-shot mutation effect prediction in a structure-aware protein language model? The answer depends on where confidence is injected. On the committed 217-assay ProteinGym benchmark, direct embedding-side modulation does not help: hard masking and soft scaling are consistently worse than baseline, and the learned per-residue gate matches baseline within noise rather than surpassing it. This rules out the simplest intuition that low-confidence structure tokens should merely be attenuated or removed before entering ProSST.

At the same time, the ensemble results show that confidence is not irrelevant. On the 214-assay intersection where multiple predictors are available, confidence-conditioned score-level gating improves over a strong flat ensemble and yields the best overall result of this study. The strongest gain appears in the lowest-confidence quartile, suggesting that pLDDT is most useful when it controls cross-model trust rather than within-model suppression.

Taken together, these findings sharpen the design lesson for confidence-aware protein modeling. Confidence should not be treated as a residue-local mask applied uniformly to pretrained structure features. It is better viewed as a model-selection or mixture-weighting signal that mediates between complementary predictors with different failure modes. Future work should therefore focus on confidence-aware routing, richer uncertainty representations, and training objectives aligned directly with variant-ranking performance.

Appendix A. Reproducibility Details

A.1 Experimental Protocol

Reproducing the reported numbers requires six stages. First, obtain the wild-type sequences, structural token sequences, and substitution tables from the upstream ProSST release. Second, obtain AlphaFoldDB structures and extract residue-aligned pLDDT arrays. Third, evaluate the ProSST-2048 baseline on all 217 assays. Fourth, evaluate the hard-mask and soft-weighting ablations using the same assay set and the same confidence arrays. Fifth, train the learned gate on a held-out protein set that does not overlap the ProteinGym assay targets, select the best validation model, and evaluate that model on the same 217 assays. Sixth, cache the per-mutant scores needed for score-level combination on the 214-assay overlap, fit both the flat and the confidence-conditioned ensembles by cross-validation, and report their assay-level summaries separately from the 217-assay direct-modulation runs.

A.2 Alignment Safeguard

Because the project studies residue-level confidence modulation, exact alignment between the confidence array and the residue sequence is essential. The release therefore treats sequence/PDB inconsistencies as explicit preprocessing errors. This choice prevents a common but under-discussed failure mode in confidence-aware pipelines: a shifted confidence array may still allow the program to run, yet the resulting ablation no longer corresponds to the intended biological positions.
