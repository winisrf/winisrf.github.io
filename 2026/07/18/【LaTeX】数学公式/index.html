14. 常见错误与建议

  1. LaTeX 大小写应统一。 正确写法是 LaTeX，而不是 Latex 或 LaTex。
  2. 不要混淆上下标。 _ 表示下标，^ 表示上标。
  3. 多字符参数应使用花括号。 推荐写 x_{ij}、x^{n+1}，不要写成 x_ij 或 x^n+1。
  4. 并非所有大写希腊字母都有命令。 例如不要使用通常无效的 \Alpha、\Beta、\Omicron。
  5. 颜色应写成 black，不是 back。 更推荐使用分组形式 \color{red}{...}，避免手动恢复颜色。
  6. 分数参数建议始终加花括号。 使用 \frac{a}{b}，而不是简写 \frac ab。
  7. 公式中的普通单词使用 \text{...}。 不要直接把长英文或中文当变量逐字排版。
  8. 微分符号建议使用直立体。 例如 \mathrm{d}x，并在前面加 \,。
  9. 绝对值和范数推荐使用语义明确的命令。 使用 \lvert x\rvert 和 \lVert x\rVert，比直接输入竖线更稳定。
  10. Markdown 平台支持程度不同。 若公式无法渲染，先确认平台是否启用了 MathJax/KaTeX，以及是否支持所使用的扩展命令。
